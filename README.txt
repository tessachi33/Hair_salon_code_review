##### _{Adds appointments for hair salon into database}, {8/28/15}_

## Description

_{This app allows stylists to view their clients and add clients to their database.}_

## Setup

* _dump the psql database into your terminal ( sql hair_salon < infile_
* _run gradle in another tab_
* _got to localhost:4567_
* _have fun being a hair slave_
* _(don't forget to run postgres while psql and bash tabs open)_



## Technologies Used

_{Java, boostarp, postgres, psql, Velocity and Spark.}_
